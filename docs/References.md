# References

Primary references for the implementation and its tests are:

1. Genest, C., Remillard, B., and Beaudoin, D. (2009). Goodness-of-fit tests for copulas: A review and a power study. *Insurance: Mathematics and Economics*, 44(2), 199--213. <https://doi.org/10.1016/j.insmatheco.2007.10.005>
2. Zhang, S., Okhrin, O., Zhou, Q. M., and Song, P. X.-K. (2016). Goodness-of-fit test for specification of semiparametric copula dependence models. *Journal of Econometrics*, 193(1), 215--233. <https://doi.org/10.1016/j.jeconom.2016.02.017>
3. Huang, W. and Prokhorov, A. (2014). A goodness-of-fit test for copulas. *Econometric Reviews*, 33(7), 751--771.
4. Breymann, W., Dias, A., and Embrechts, P. (2003). Dependence structures for multivariate high-frequency data in finance. *Quantitative Finance*, 3(1), 1--14.
5. The CRAN package `gofCopula`, version 0.4-3, by Simon Trimborn, Ostap Okhrin, and Martin Waltz, is the behavioral reference for this migration.
